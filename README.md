## Quick Start to Setup

1. Clone the repo:

```bash
git clone https://github.com/RevoU-FSSE-4/milestone-1-bintangsenjapratama.git
```

2. Create new branch named "dev"

```bash
git branch -b dev
```

3. Checkout to "dev"

```bash
git checkout dev
```
